# actividad-semana3-java

## Descripción
Repositorio correspondiente a la Actividad Sumativa de la Semana 3, en la cual se desarrolla un programa en Java utilizando compilación por línea de comandos (javac) y ejecución mediante la máquina virtual de Java (JVM).

El programa permite ingresar datos de un vehículo por medio de consola y mostrarlos en pantalla. Además, se documenta el caso de estudio de una empresa distribuidora de alimentos, considerando sus requerimientos, historias de usuario y planificación del desarrollo.

---

## Requerimientos Funcionales
- Permitir ingresar la marca del vehículo
- Permitir ingresar el modelo del vehículo
- Permitir ingresar la cilindrada del vehículo
- Permitir ingresar el tipo de combustible
- Permitir ingresar la capacidad de pasajeros
- Mostrar los datos ingresados por el usuario en pantalla

- Permitir calcular el costo de despacho según el monto de compra
- Aplicar despacho gratis para compras sobre $50.000 dentro de un radio de 20 km
- Aplicar tarifa de $150 por kilómetro para compras entre $25.000 y $49.999
- Aplicar tarifa de $300 por kilómetro para compras menores a $25.000

---

## Requerimientos No Funcionales
- El programa debe ejecutarse en la JVM
- Debe funcionar mediante consola (línea de comandos)
- Debe ser portable entre distintos sistemas operativos
- Debe ser fácil de usar y entender
- El código debe estar documentado con comentarios claros
- La información ingresada debe procesarse correctamente

---

## Historias de Usuario
- Como usuario, quiero ingresar los datos de un vehículo para visualizarlos en pantalla
- Como usuario, quiero ingresar el monto de mi compra para saber el costo del despacho
- Como usuario, quiero conocer si tengo despacho gratis según mi compra
- Como usuario, quiero ingresar la distancia en kilómetros para calcular el costo del envío
- Como usuario, quiero ver el resultado del cálculo de despacho de forma clara
