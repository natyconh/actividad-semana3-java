# actividad-semana3-java

## Descripción
Repositorio correspondiente a la actividad sumativa de la semana 3, donde se desarrolla un programa en Java utilizando compilación por línea de comandos.

## Requerimientos Funcionales
- Permitir ingresar nombre del usuario
- Permitir ingresar edad del usuario
- Mostrar los datos ingresados en pantalla

## Requerimientos No Funcionales
- El programa debe ejecutarse en la JVM
- Debe funcionar mediante consola
- Debe ser portable en distintos sistemas operativos

## Historias de Usuario
- Como usuario quiero ingresar mi nombre para visualizarlo en pantalla
- Como usuario quiero ingresar mi edad para verificar mis datos
- Como usuario quiero ver mis datos ingresados para confirmar la información

## Cronograma

| Día | Actividad |
|-----|----------|
| Día 1 | Desarrollo del código |
| Día 2 | Compilación y pruebas |
| Día 3 | Documentación e informe |
